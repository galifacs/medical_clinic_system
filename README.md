# medical_clinic_system
Medical Clinic Management System developed as part of a Database Systems course using PHP and MySQL.

Features

User authentication and authorization
Patient registration
Patient profile management
Medical visit scheduling
Role-based access control (Admin / Patient)
AES encryption of sensitive patient data
Password hashing using PHP password_hash()
Dark / Light theme support

Technologies

PHP
MySQL
HTML/CSS
XAMPP

Installation

Clone the repository
Import medical_clinic.sql
Rename db_example.php to db.php
Configure database credentials
Run the application using XAMPP

User Roles

Admin

View all patients
Register patients
Schedule visits
Delete patients
View all visits

Patient

View own profile
View own visits
Register account

Author

Student project created for educational purposes.
